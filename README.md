# coisini-blog-admin


## 介绍
> coisini-blog-admin 是基于 vue3 + element-plus 搭建的博客后台管理系统

### 安装 yarn
```sh
npm install yarn
```

### 安装依赖
```sh
yarn
```

### 启动项目
```sh
yarn dev
```

### 打包项目
```sh
yarn build
```
